# DotnetDevkit.Cache.Abstractions

This package contains the cache abstractions and types for
`DotnetDevkit.Cache` [link to docs](../DotnetDevkit.Cache/README.md).

## Install

```bash
dotnet add package DotnetDevkit.Cache.Abstractions
```

## Usage
Using this package, you can depend on the `ICache` interface in your services
without needing to reference the concrete cache implementation.
